# 智慧藥櫃盤點 GitHub Pages 範例

這個資料夾可直接上傳到 GitHub repository，並用 GitHub Pages 發佈。

## 檔案

- `index.html`：入口網站與入口清單管理器。
- `cabinets.json`：入口網站讀取的藥櫃清單。
- `oral-1.html`：口服藥 1 盤點頁。

## 新增藥櫃

1. 先在盤點頁後台設定好藥櫃。
2. 下載該藥櫃離線檔，改成適合的檔名，例如 `oral-2.html`。
3. 把新的 HTML 上傳到 GitHub repository。
4. 打開入口頁的「入口清單管理」。
5. 新增藥櫃名稱、檔案名稱、分類。
6. 下載新的 `cabinets.json`。
7. 上傳並覆蓋 GitHub repository 裡的 `cabinets.json`。

GitHub Pages 更新後，入口頁就會顯示新增的藥櫃。
